# Tutorial media assets

Put tutorial screenshots and short clips here so they render on /tutorial.

Naming (matches step IDs in app/api/misc.py):
- code.png / code.mp4
- login.png / login.mp4
- terms.png / terms.mp4
- select.png / select.mp4
- upload.png / upload.mp4
- view.png / view.mp4

Guidelines
- Clips: 1–15 seconds, no audio, 16:9 preferred, < 10MB ideal.
- Images: PNG or JPG, ~1280×720 recommended.
- Keep file names lowercase; provide both image and clip where helpful; the page will show the best available.

Tips for capture
- Windows: Win+Alt+R (Xbox Game Bar) or PowerPoint screen recorder.
- macOS: Shift+Cmd+5 screen capture.
- Trim with built-in Photos/QuickTime or free tools.

Once added, reload /tutorial; the placeholder will be replaced by your media.
